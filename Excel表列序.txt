给定一个Excel表格中的列名称，返回其相应的列序号。
例如，

    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
示例 1:
输入: "A"
输出: 1

public int titleToNumber(String s) {
        char[] ch = s.toCharArray();
        int ret = 0;
        for(int i = 0; i < ch.length; i++){
            ret = ret * 26 + (ch[i] - 'A' + 1);
        }
        return ret;
    }
